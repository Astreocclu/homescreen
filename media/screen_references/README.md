## ğŸ“¸ Image Requirements

### Quality Standards
- **Resolution**: Minimum 800x600, preferably 1200x800 or higher
- **Format**: JPG or PNG
- **Focus**: Clear, well-lit images showing screen texture and pattern
- **Angle**: Straight-on view of the screen mesh pattern

### Content Guidelines
- **Close-up shots** showing mesh pattern detail
- **Installed examples** showing screens on actual windows/doors
- **Different lighting conditions** (indoor, outdoor, various times of day)
- **Multiple angles** for each screen type

### Naming Convention
- Use descriptive names: `security_fine_mesh_closeup.jpg`
- Include brand names if applicable: `phifer_suntex_90.jpg`
- Add lighting info: `solar_screen_backlit.jpg`

## ğŸ¤– How AI Uses These References

### Image Analysis
1. **Pattern Recognition**: AI analyzes mesh patterns, density, and texture
2. **Color Matching**: Learns realistic color variations for each screen type
3. **Lighting Effects**: Understands how screens interact with light
4. **Installation Context**: Sees how screens look when properly installed

### Generation Process
1. **Reference Selection**: AI picks the most appropriate reference image
2. **Pattern Extraction**: Analyzes the mesh pattern and characteristics
3. **Adaptive Application**: Applies pattern to detected windows/doors
4. **Lighting Adjustment**: Matches lighting conditions of the target image
